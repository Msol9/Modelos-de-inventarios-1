Programa para optimizar la gestión de inventarios utlizando: el modelo clásico y el modelo con descuentos por cantidad, el cual permite calcular la cantidad económica de pedido, 
el costo total y elegir la mejor opción entre precios fijos o con descuentos. Con el objetivo de ayudar a las empresas a reducir costos y tomar decisiones más eficientes en sus
compras y almacenamiento.
